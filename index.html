Seed phrase
Each plasma fractal is randomly generated starting with a "seed" value. If you get a pattern you like, you can get the same result by starting with the same seed. To make it easier to save and share seed values, you can input any text phrase as a seed value.
Parameters

The algorithm that generates a plasma fractal starts with a smooth gradient based on randomly chosen color values for each of the four corners. It then perturbs this gradient in a recursive process where each step randomly "displaces" the color value in the center of the gradient, divides the gradient into four quadrants, and applies the recursion to each quadrant.

Three parameters control the Gaussian distribution used to choose the random displacement.

σ0:
The standard deviation used for the initial random displacement. Larger values make large displacements more likely, resulting in bigger peaks and valleys.
μ0:
The mean used for the initial random displacement. Positive values bias the result in favor or using higher color values, and lower values do the opposite. Using 0 results in the most even overall distribution of colors.
Attenuation:
Each level of recursion attenuates the standard deviation and mean values used for the random choice of displacement, so that the algorithm starts with large changes and finishes with small adjustments. Stronger attenuation results in a smoother fractal, and weaker attenuation results in a bumpier fractal. Note that the scale is reversed, and higher values for this parameter result in less attenuation.
